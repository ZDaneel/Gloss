# Theris — 部署指南

## 环境要求

| 依赖 | 版本要求 |
|------|---------|
| Node.js | ≥ 18.12 |
| pnpm | 最新版 |
| Python | ≥ 3.10 |
| PostgreSQL | 16 |

> 可以使用 `nvm` 管理 Node.js 版本：`nvm use 18`

---

## 1. 数据库初始化

创建数据库并执行初始化脚本：

```bash
createdb paper-project
psql -d paper-project -f dbscript/init.sql
```

---

## 2. 模型服务（`MockProject/`）

> `MockProject` 是模型服务的本地模拟，仅用于开发和集成测试。
> 生产环境请替换为真实模型服务，并将 `MOCK_URL` 指向对应地址。
> 真实服务需实现的接口协议，请参考主 README 的 [Model Service](../README.md#model-service) 章节。

```bash
cd MockProject
pip install -r requirements.txt
python main.py
# 运行在 http://localhost:3008
```

---

## 3. 后端服务（`PaperProject/`）

复制环境变量文件并按需修改：

```bash
cd PaperProject
cp .env.example .env
```

```env
DATABASE_URL=postgresql://postgres:123456@localhost:5432/paper-project  # 数据库连接地址
MOCK_URL=http://localhost:3008      # 模型服务地址
UPLOAD_DIR=./uploads                # 文件上传目录
PARAGRAPH_TIMEOUT=60                # 段落检索超时（秒）
CHAT_TIMEOUT=60                     # 对话超时（秒）
```

安装依赖并启动：

```bash
pip install -r requirements.txt
python run.py
# 运行在 http://localhost:3007
```

---

## 4. 前端（`paper-project/`）

复制环境变量文件并按需修改：

```bash
cd paper-project
cp .env.example .env
```

```env
NUXT_GLOB_API_URL=http://localhost:3007   # 后端服务地址
PARAGRAPH_TIMEOUT=60000                   # 段落检索超时（毫秒）
CHAT_TIMEOUT=60000                        # 对话超时（毫秒）
```

安装依赖并启动：

```bash
pnpm install

# 开发模式
pnpm run dev
# http://localhost:3006

# 生产构建
pnpm run build
pnpm run preview
```
